2. Tool for web developers

這個是最重要的部分，如果可以畫星星，就多畫一點，
Google為了方便這些網頁的開發人員(?)，所以當你在開網頁的時候，點滑鼠右鍵，會跑出兩個超重要的選項，

- 一個是檢視網頁原始碼(Ctrl+U)
原始碼可以說就是這個網頁的html file，有些人會把全部的東西都塞在這裡，也有人會分css、js檔，你就可以從這裡去看每個人的寫法，詳細的應用上，在爬蟲的時候，非常方便，只要查同一個標籤的資料有沒有你要的，全部爬下來就好。

- 一個是檢查(Ctrl+Shift+I)
檢查就是另一個更厲害的功能了，單純點檢查，可能看不出什麼端倪，
但是如果你將自己想要查詢的字反白，再點右鍵，點檢查，就會發現原來這段文字在網頁原始碼的這個地方啊!運用這些標籤啊!再來旁邊還會顯示他運用的css(也可能沒有)，如果有的話，可以直接在這裡修改顏色樣式，不過只是測試，真的修改還是要去文字編輯器修改喔。

改html其實非常繁瑣，有想要的功能或是不知道用哪個標籤，就趕緊Google，而我的習慣是，同時打開index.html(Google會打開index.html，不過不是網址，而是你的本地端檔案地址)，和Brackets來修改index.html，當我修改一段文字或是一個效果的時候，我就把網頁更新一次，看改出來的效果如何，避免一次改太多。
